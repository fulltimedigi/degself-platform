# مفهوم شعار degself

## الفلسفة

**degself** = دق سلف = "ضغطة زر تشغّل سيارتك" = الحل السهل لمشاكل السيارة

الشعار يجب أن يعكس:
1. **مفتاح ريموت حديث** (smart key) - عصري ومناسب لكل السيارات
2. **بساطة وعصرية** - يعمل في كل الأحجام (favicon → billboard)
3. **هوية كويتية** - الأصفر والأسود يذكر بالسيارات الرياضية والـ taxi
4. **رمز التشغيل** - الزر الدائري في وسط المفتاح = "دق" = start
