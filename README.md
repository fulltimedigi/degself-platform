# degself — منصة ورش الكويت

> دق سلف — أول منصة ذكية تربط أصحاب السيارات بورش الصيانة في الكويت

## نظرة عامة

**degself** هي منصة PWA متخصصة في:
- 🔍 تشخيص ذكي بالـ AI لمشاكل السيارة
- 📍 خريطة تفاعلية لـ **1802 ورشة** موثقة في 6 محافظات
- 📅 حجز موعد مباشر مع الورشة
- 💬 تواصل عبر واتساب
- ⭐ تقييمات وآراء حقيقية من Google
